# AtyrauMonitor

Локальный мониторинг публикаций по Атырау и Атырауской области.

## Ручной запуск

```powershell
.\run_monitor.ps1
```

## Состояние

- `state\monitor_state.json` — последние успешные срезы лент и служебные отметки.
- `state\SENT_CANONICAL.txt` — постоянный список уже отправленных canonical URL.
- `state\monitor.log` — технический лог запусков.

Секреты Telegram читаются из `.env`.

## Защита от шума

- Основной список источников соответствует исходному ТЗ; дополнительные RSS/sitemap/search/API используются только внутри этих доменов.
- Внешний поиск выполняется через RSS-выдачу новостей по заданным запросам; домены из основного списка при этом пропускаются, чтобы не дублировать основной мониторинг.
- Уже отправленные canonical URL и служебные страницы отбрасываются до скачивания статьи.
- Дополнительные кандидаты ограничены, чтобы цикл не растягивался дольше 15 минут.
- Материалы старше 600 минут не отправляются; подозрительные даты из будущего нормализуются под время Атырау.
